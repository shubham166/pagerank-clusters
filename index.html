    Hello

    This is my small academic project on Efficient Page Rank computation on clusters. It uses message passing interface for parallel computation by scattering chunks of pages to individual nodes and gathering the result at master node. Similarly, openMp APIs are used to utilize all the cores of a node efficiently and give result taking minimum time.
